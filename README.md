## 💡 Description

**Push_swap** est un projet d'algorithmique dont le but est de trier des données sur une pile, avec un set d'instructions très limité, et en un minimum de coups.
Le but est de calculer et d'implémenter un algorithme de tri optimisé (ex: Radix, Turk algorithm, ou un Quicksort adapté) pour gérer de grandes quantités de nombres.

## 🛠️ Compétences acquises
* **Algorithmique & Complexité :** Trouver la solution la plus courte et la plus rapide.
* **Structures de données :** Manipulation avancée de piles (Stacks) et de listes chaînées.
* **Optimisation :** Réduire le nombre d'opérations au strict minimum.

## 🚀 Utilisation

Compiler le projet :
```bash
make
```
